// CRIANDO UM PDF

1* - Instalar o iTextSharp (Gerenciador de Solucoes -> Referencias 'botao direito' -> Gerenciar Pacotes do Nuget -> Intalar)

2* - implementar as bibliotecas no forms:
iTextSarp;
iTextSarp.text;
iTextSarp.text.pdf;

3* - criar o arquivo PDF:
.criar em uma string o caminho aonde o arquivo sera armazenado + nome do pdf
.criar um FileStream (criador de arquivo) com a string criada anteriormente, o tipo de arquivo
.instanciar um objeto da Classe Document com o PageSize.A4 ou outro desejado
.criar um escritorPDF da classe PdfWriter que recebbe o PdfWriter.GetInstance(Documento instanciado, FileStream instanciado)

4* - criar o paragrafo
.criar uma string vazia
. instanciar um objeto 'paragrafo' da classe Paragraph com os paramentros:

 => string_criada_vazia, new iTextSharp.text.Font(iTextSharp.text.Font.NORMAL, 14, (int)System.Drawing.FontStyle.Bold)

.definir o alinhamento do paragrafo (paragrafo.Alignment = Element.'tipo_de_alinhamento')
.adicionar o texto

5* - inserir o paragrafo no documento
.doc.Open();
.doc.Add(paragrafo);
.doc.Close();

///////// mudar a font da linha (formatacao)
ex:
paragrafo.Font = new iTextSharp.text.Font(iTextSharp.text.Font.NORMAL, 14, (int)System.Drawing.FontStyle.Italic);
